How It Works

Carnival Day Workflow
  1. Event called (e.g., "100m 8/9 girls")
  2. Go to Results tab → Record results
  3. 1st and 2nd place auto-qualify for district
  4. When carnival ends → Export Top 2
  5. Import CSV into District Carnival system
